# Weibo
根据微博的接口集成的微博客户端

    这个项目是根据微博的开放平台中的接口文档集成的一个项目。其中主要以登录页和首页为主，
    中间的➕页和我页实现了一部分的功能。主要是对数据的处理和接口的衔接方面的应用。
